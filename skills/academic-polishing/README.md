# Academic Polishing

独立可用的学术文体打磨技能。适用于 CS/AI/ML 论文段落的 prose 质量提升、去AI化改写和 claim 强度控制。

## 触发方式

- "帮我把这段 Method 润色成论文口吻"
- "去AI化改写这段 Introduction"
- "这段 claim 太强了，帮我降强度"
- "polish prose quality"

## 核心能力

| 能力 | 说明 |
|------|------|
| Prose Quality Gate | 检查提纲句、元评论、代码讲解口吻等常见问题 |
| 去AI化改写 | 替换 AI 典型连接词、空洞形容词，建立深层逻辑衔接 |
| Claim 强度控制 | 按证据等级匹配 Strong/Medium/Weak 三档表述 |
| Method 叙事强化 | 将公式罗列改写为"问题→设计→机制→收益/边界"叙事 |

## 与其他 Skill 的关系

- 被 `academic-paper-writer` 在 Steps 6.3-6.5（Prose 质量闸门）中委托调用
- 可独立使用于任何需要润色的学术段落
