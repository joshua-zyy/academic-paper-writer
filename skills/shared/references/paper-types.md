# 论文类型定义

## 类型枚举

| 类型 | 说明 | 默认结构 |
|------|------|---------|
| `empirical` | 实验性论文，提出方法+实验验证 | IMRaD 变体 |
| `theory` | 理论性论文，提出定理/理论分析 | 理论结构 |
| `survey` | 综述论文，系统梳理领域工作 | Survey 结构 |
| `reproducibility` | 复现报告，验证已有结果 | 复现结构 |
| `position` | 立场论文，提出观点和争论 | 立场结构 |

## 判断方法

1. 检查仓库中是否存在实验代码/数据 → empirical
2. 检查是否有数学证明或理论分析 → theory
3. 检查是否以总结他人工作为主 → survey
4. 检查是否旨在复现他人结果 → reproducibility
5. 其余 → position（以论证和观点为主）
