# QA Contract: 交付前 QA 检查清单

`chart-from-data` 模式的 Step 7 中使用。逐项检查，所有项 pass 方可交付。

## 检查清单

### 1. 核心结论可见性
- [ ] Core conclusion 在图上一目了然？
- [ ] 无冗余面板（每个 panel 回答一个独特问题）？

### 2. 数据完整性
- [ ] 所有数据来源已确认（无虚构数据点）？
- [ ] 源数据文件已随图交付（CSV/TSV）？
- [ ] 离群点已标注或说明？

### 3. 统计信息
- [ ] 误差棒标注了含义（std / SEM / 95% CI）？
- [ ] 样本量（n）已在图注中说明？
- [ ] 统计检验方法已在图注中说明？
- [ ] 显著性标注使用标准符号（* p<0.05, ** p<0.01, *** p<0.001）？

### 4. 可读性
- [ ] 文字在最终缩放尺寸下可读（最小 7 pt）？
- [ ] Panel label 位置正确（12 pt bold，左上角）？
- [ ] 坐标轴标签完整且有单位？

### 5. 配色
- [ ] 色盲友好（不以纯红/纯绿作为唯一区分）？
- [ ] 灰度打印友好（亮度差 > 30%）？
- [ ] 无 rainbow / jet 色板？
- [ ] 颜色数量不超过 6 种？

### 6. 导出规范
- [ ] SVG 文字可编辑（fonttype="none" check passed）？
- [ ] PDF 选项正确（fonttype=42）？
- [ ] 位图版本 dpi >= 300？

### 7. 伦理合规
- [ ] 坐标轴未截断（或已标注截断标记）？
- [ ] 未使用 3D 效果展示 2D 数据？
- [ ] 未选择性展示有利结果？
- [ ] 无隐藏的负面结果？
