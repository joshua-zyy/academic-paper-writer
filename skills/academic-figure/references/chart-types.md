# Chart Types: CS/AI/ML 实验图表类型详解

`chart-from-data` 模式中图表类型选择与生成的详细参考。

## 1. 训练/验证曲线

### 适用场景
- 模型收敛性分析（loss 随 epoch 变化）
- 多方法训练动态对比
- Early stopping / overfitting 判断

### 数据格式
```csv
epoch,method_a_loss,method_a_std,method_b_loss,method_b_std
1,0.85,0.02,0.92,0.03
2,0.72,0.015,0.81,0.025
...
```

### 布局建议
- x 轴：Epoch（从 0 或 1 开始）
- y 轴：Loss / Accuracy / AUC
- 每种方法一条线 + 半透明 std 置信带
- 最佳 epoch 处添加竖虚线 + 标注
- 若 y 轴起点非 0，用截断标记

### 常见错误
- 只有均值线没有置信带（掩盖方差）
- 颜色过多无法区分（超过 6 种方法建议分两图）
- 坐标轴范围导致视觉误导（如只放大顶部 5% 区域）

## 2. 消融实验（Ablation Study）

### 适用场景
- 验证每个组件的贡献
- 从满模型逐步移除组件观察性能变化

### 数据格式
```csv
variant,performance,std
full_model,92.5,1.2
w_o_attention,88.3,1.5
w_o_residual,87.1,1.4
w_o_augmentation,90.2,1.3
```

### 布局建议
- x 轴：从左到右 = 满模型 → 逐一移除组件（颜色由深到浅）
- 满模型用最深色，移除越多的组件颜色越浅
- 使用 hatching 纹理辅助灰度打印区分
- 标注相对于满模型的性能下降量

### 常见错误
- 颜色顺序混乱（应保持满模型在左、逐步去除在右）
- 缺少 error bar（消融实验通常有多轮运行）
- 没有标注统计显著性

---

## 3. 性能对比图

### 适用场景
- 多数据集 × 多方法的性能对比
- Benchmark 结果汇总

### 数据格式（长格式）
```csv
dataset,method,score,std
CIFAR-10,Ours,97.2,0.3
CIFAR-10,Baseline_A,95.1,0.4
CIFAR-10,Baseline_B,94.8,0.5
ImageNet,Ours,78.5,0.6
...
```

### 布局建议
- 分组柱状图：x 轴 = 数据集，组内 = 各方法
- 或热力图：x 轴 = 方法，y 轴 = 数据集，颜色 = 性能
- 每组内方法顺序保持一致（ours 放最左或最右）
- 最优值加粗 / 加星号标注

### 常见错误
- 用 3D 柱状图（遮挡数据，难以准确读取）
- 跨数据集比较时尺度不一致
- 没有报告方差

---

## 4. 混淆矩阵

### 适用场景
- 分类结果细节分析
- 类别级错误模式识别

### 数据格式
```python
matrix = np.array([[980, 12, 8],
                   [15, 952, 33],
                   [5, 21, 974]])
classes = ["cat", "dog", "bird"]
```

### 布局建议
- 对角线高亮（深色）
- 非对角线用浅色渐变
- 每个格子标注数值
- 行列标签 + 总 accuracy

### 常见错误
- 显示归一化矩阵但未标注（需明确是 counts 还是比例）
- 类别顺序混乱（应与论文描述一致）
- 小数值被四舍五入掩盖（< 0.5% 的情况用 " <0.1" 而非 0）

---

## 5. t-SNE / UMAP 散点图

### 适用场景
- 特征表示质量的可视化
- 不同类别的 embedding 分布

### 数据格式
```python
embeddings = np.load("tsne_results.npy")  # (n, 2)
labels = np.load("labels.npy")            # (n,)
```

### 布局建议
- 每种类别一种颜色，图例标注
- 透明度 0.5-0.7 避免重叠遮盖
- 点大小 3-8（取决于样本量，大样本用小点）
- 可添加轮廓线或 convex hull 强化聚类边界

### 常见错误
- 对 t-SNE 结果过度解读（t-SNE 保留局部结构但破坏距离）
- 没有标注 perplexity / 随机种子
- 点过大导致遮挡严重

---

## 6. 箱线图 / 小提琴图

### 适用场景
- 多轮实验的结果分布比较
- 方法稳定性和方差分析

### 布局建议
- 箱线图：中位数线 + 四分位框 + 须 + 离群点
- 小提琴图：箱线图 + 密度估计（适合小样本）
- 同一方法的不同条件用同色系渐变

### 常见错误
- 样本量小（< 5）时用箱线图产生误导
- 没有标注统计检验方法
- 离群点过多时忽略分析

---

## 7. 雷达图

### 适用场景
- 多维度综合对比（精度、速度、参数量、泛化性等）
- 需要展示方法在不同指标上的权衡

### 布局建议
- 维度 3-8 个为宜
- 每个维度归一化到 [0, 1] 或同尺度
- 不同方法用不同颜色 + 半透明填充
- 图例放在图外

### 常见错误
- 维度过多导致图形难以解读
- 未归一化导致某一维度主导视觉
- 忽略量纲差异

---

## 8. 森林图 / 点范围图

### 适用场景
- 多数据集上的效果汇总
- Meta-analysis 风格的结果展示

### 布局建议
- 每行一个任务/数据集
- 点 = 均值，横线 = 置信区间
- 垂直线 = baseline 参考值
- 最优值用菱形标记

### 常见错误
- 置信区间含义未标注
- 行顺序没有按效应量排序
- 垂直线位置不合理
